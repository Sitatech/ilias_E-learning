Session-Statistiken-Stats
Anzahl der Sessions: Anzahl aller Sessions die aus der Datenbank konstruierbar sind.
Auswertbare Sessions: Anzahl aller Sessions die auswertbar, also messbar hinsichtlich der Verweilzeit, sind.
Anzahl beteiligter Benutzer: Anzahl der User für die Einträge im Usertracking vorhanden sind.
Auswertbare Benutzer: Anzahl der User die an den auswertbaren Sessions teilnahmen.
Auswertbare Sessionzugriffe: Auswertbare Sessionzugriffe sind Zugriffe die innerhalb einer Session stattfanden (ab den zweiten getätigten Zugriff aufgezeichnet). Diese werden für alle Sessions aufsummiert.
durchschn. Seitenzugriffe je Session: Anzahl der Zugriffe die die User innerhalb einer Session durchschnittlich getätig haben.
Gesamtdauer aller Sessions: Die Dauer aller Sessions wird aufsummiert.
durchschn. Sessionlänge: Zeit die ein User durchschnittlich während einer Session verbracht hat.
Längste Session: Die 10 längsten Sessions werden hier mit der jeweilgen Gesamtzeit aufgelistet.
Längste Gesamtdauer pro Benutzer: Die User, die sich die meiste Zeit im Lernmodul aufgehalten haben.
Meiste Sessions: Die User, die die meisten Sessions innnerhalb des Lernmoduls inne hatten.
